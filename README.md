pydaruma
========

### Introdução

Integração com impressoras Daruma utilizando a biblioteca DarumaFramework.dll e libDarumaFramework.so

### Dependência
* Python 3.4.2 ou superior
* DarumaFramework.dll (Para Windows) ou libDarumaFramework.so (Para linux)
